# Haskell

## First steps

1. Installing on Ubuntu

```
$ sudo apt install haskell-stack
$ sudo stack upgrade
$ stack ghci
ctrl+Z
```

2. Create a new project with `stack`

```
$ stack new haskell-for-good
$ cd haskell-for-good
$ stack build
$ stack test
```
